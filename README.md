###OuShi 小游戏（多关卡式）

##项目名称更新为：ORGAN-C

###构思
人物进入场景，然后oushi，玩家通过操作，让人物在规定时间内完成并离开
操作：
- 快速点击屏幕加速
- 拿走手机
- 吃药

非点击期间，无倒计时，仅仅展示自动动画
类点击泰坦或者 Cookie clicker
